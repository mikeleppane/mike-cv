### Prerequisites

- [Rust](https://www.rust-lang.org/) (latest stable version)
- [Cargo Leptos](https://github.com/leptos-rs)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/mike-cv-in-rust.git
cd mike-cv-in-rust
```

2. Install Cargo Leptos:
